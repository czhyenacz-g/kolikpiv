# Kolik piv to je? 🍺

Minimální Next.js aplikace pro výpočet, kolik piv něco stojí.

## Instalace

```bash
npm install
```

## Spuštění

```bash
npm run dev
```

Aplikace běží na [http://localhost:3000](http://localhost:3000)

## Funkce

- Výpočet ceny v pivech
- Výpočet odpracovaných hodin
- Uložení ceny piva a mzdy do localStorage
- Responzivní design
- Tmavý vzhled

## Použitá technologie

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
